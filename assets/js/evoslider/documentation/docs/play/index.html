play(interval)


Version: Pro

play(interval)

returns Evo Slider

Starts the automatic slideshow and sets the interval time in milliseconds. If no interval is given, default is used (5000).

Example:

<script type="text/javascript">
    var myEvoSlider = $("#mySlider").evoSlider();
    
    // Starts slideshow
    myEvoSlider.play(5000);
</script>
