Using Dailymotion's Service

The Evo Slider plugin allows you to display videos from Dailymotion.

You can use the external option to load data from Dailymotion.

Dailymotion parameters:

Name Type Description
data String Set to "dailymotion" when you want to load content from Dailymotion.
source String Possible values are: "videos", "favorites", "features", "subscriptions", "playlist", "contest", "channel", or "group".
user_id String Dailymotion user id.
group_id String Dailymotion group id.
channel_id String Dailymotion channel id.
contest_id String Dailymotion contest id.
playlist_id String Dailymotion playlist id.
title_length Number Set the number of characters in the title.
desc_length Number Set the number of characters in the text description.

Examples:

Retrieve all of a user's recent videos:

external: {
    data: "dailymotion",
    source: "videos",
    user_id: "dailymotion_user_id"
    }

Retrieve all of a user's favorite videos:

external: {
    data: "dailymotion",
    source: "favorites",
    user_id: "dailymotion_user_id"
    }
