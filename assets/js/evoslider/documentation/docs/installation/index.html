Installing The jQuery Library

EvoSlider is built on top of the jQuery JavaScript framework and you must include this in order for the EvoSlider to work. A number of large enterprises like Google provides hosted copies of jQuery. You can also download your own copy of jQuery library and host it yourself if you wish.

Visit jQuery website and click the download button. Save the file on your hard drive somewhere you will be able to access it later.

Installing jQuery Easing Plugin

Visit jQuery Easing Plugin website to download the file. Save the file on your hard drive somewhere you will be able to access it later.

Installing Evo Slider Plugin

Unzip the Evo Slider download package you received when you purchased the Evo Slider. If you no longer have it, you can redownload it from your CodeCanyon account.

Save the Evo Slider files somewhere on your hard drive you'll be able to access later.
