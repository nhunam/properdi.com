youtube


Version: Pro

youtube

type: Object

default: { autoplay: 0, showinfo: 1, autohide: 2 }

You can customize the YouTube player using the YouTube player parameters.

You can see the list of all available parameters here: https://developers.google.com/youtube/player_parameters#Parameters

Example:

var myEvoSlider = $("#mySlider").evoSlider({
    youtube: {
        autoplay: 0,
        showinfo: 1,
        autohide: 2
    }
});
