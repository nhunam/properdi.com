onpause


Version: Pro

onpause

Triggers when the slider slideshow stops playing.

Example:

<script type="text/javascript">
    var myEvoSlider = $("#mySlider").evoSlider();
    
    // Bind function to the onpause event
    myEvoSlider.bind("onpause", function(){
        alert("Evo Slider stops playing");
    });
</script>
