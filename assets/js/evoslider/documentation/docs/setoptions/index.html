setOptions(key, value)


Version: Pro

setOptions(key, value)

returns Evo Slider

Manipulates the Evo Slider options at run time.

Example:

<script type="text/javascript">
    var myEvoSlider = $("#mySlider").evoSlider();
    
    // Sets interval option
    myEvoSlider.setOptions("interval", 10000);
</script>
