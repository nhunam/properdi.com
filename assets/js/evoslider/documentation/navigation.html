  • API Methods
    • destroy() Destroy Evo Slider instance
    • getIndex() Returns the current index
    • getLength() Returns the slides length
    • getNext() Returns the next index
    • getOptions() Gets options value
    • getPrev() Returns the previous index
    • next() Displays the next slide
    • pause() Stops slideshow
    • play() Starts slideshow
    • playToggle() Starts or stops slideshow
    • prev() Displays the previous slide
    • resize() Changes the width and height
    • setOptions() Sets the Evo Slider options at run time
    • show() Show the specified slide

    • bind() Bind a callback function to an Evo Slider event

    • onplay Triggers when the Evo Slider starts playing
    • onstart Triggers before slide transition
    • trigger() Manually trigger an Evo Slider event
    • unbind() Removes all functions attached to an Evo Slider event
