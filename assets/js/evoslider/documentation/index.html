jQuery Evo Slider Documentation

Welcome to Evo Slider Documentation

Evo Slider is a jQuery slider with slideshow, multiple layout, CSS skinning and much more features. Saves development time - Powerful features - Simple implementation - Fully customizable.

With Evo Slider you can easily create amazing photo galleries or html content slider. You can choose how to display your slider, in accordion, fading slider, or scrolling mode. See all features here.

We have provided a solid documentation for the Evo Slider Pro in this file, so take your time to explore it if you want to learn more about how to customize the Evo Slider. The most basic stuff about Evo Slider can be found in the Installation and in the Quick start Reference. Make sure you read these tutorials before you install Evo Slider.

If after exploring the documentation you don't find the answer, you can find or post your question in the discussion section of the Evo Slider in the CodeCanyon. Or, contact us here: help@evoslider.com.

Keyboard UsageSpeed it up.

The keys are working if the search field has focus or your mouse cursor is in the navigation area. ESC key is global. 

UP      - Previous category / Previous item
DOWN    - Next category / Next item
ENTER   - Open category / Load selected item
ESC     - Clear search field and set focus
